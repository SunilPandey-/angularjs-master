Order By Filter only Works with an Array or array-like object (e.g. NodeList, jQuery object, TypedArray, String, etc)
You can change the sorting order by setting reverse to true. By default, items are sorted in ascending order. The comparison is done using the comparator function. If none is specified, a default, built-in comparator is used ( in a nutshell, it compares numbers numerically and strings alphabetically).
orderBy: 'name':true
